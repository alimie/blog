Memulakan Kembali Servis Rekaan Grafik dan Pembangunan Laman Web

Saya mempunyai satu perniagaan jenis perkhidmatan yang berdaftar di bawah SSM dengan nama AMZG Resources. Walaupun telah lama didaftarkan, namun bisnes ini tidaklah saya jalankan dengan bersungguh-sungguh.

Antara perkhidmatan yang saya tawarkan melalui bisnes ini ialah perkhidmatan membina laman web serta khidmat reka grafik. Bagi perkhidmatan membina laman web, saya akan membantu membangunkan laman web yang dikehendaki oleh klien samada laman web korporat, blog ataupun web e-commerce.

Bagi perkhidmatan reka grafik pula, saya pernah menyediakan servis rekaan web banner (untuk dipaparkan di laman web) serta rekaan bunting dan banner yang boleh dicetak klien di kedai-kedai percetakan.

Saya dalam proses melengkapkan portfolio contoh-contoh rekaan grafik dan laman web yang dibangunkan, sebagai rujukan kepada sesiapa saja yang berminat untuk mendapatkan kedua-dua jenis perkhidmatan daripada saya ini.
